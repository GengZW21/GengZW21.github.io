    圆周率的计算

    python python
    发布时间 : 2020-11-27 13:47
    阅读 :

    1.公式法：

    pi=0
for i in range(100):
    pi += 1/pow(16,i)*(\
        4/(8*i+1)–2/(8*i+4)–\
        1/(8*i+5)–1/(8*i+6))
print("圆周率的近似值是:{:.9f}".format(pi))
    圆周率的近似值是：3.141592654

    2.蒙特卡罗方法：

    from time import perf_counter
darts =1000*1000
hits = 0
start = perf_counter()
for i in range(0,darts):
    x,y=random(),random()
    dist = pow(x**2+y**2,0.5)
    if dist <= 1:
        hits += 1
print("圆周率的近似值是:{}".format(pi))
print("运行时间是:{:.5f}s".format(perf_counter()–start))
    圆周率的近似值是：3.141592653589793
运行时间是：1.28212s
